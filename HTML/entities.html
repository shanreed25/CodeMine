Entities

Reserved characters, such as <,>, ", and &, in HTML are have special meaning in a HTML document. If these characters are used within the content of the page, browsers may misinterpret them as part of the tags.

Use HTML Entities to escape certain characters so you can add HTML code as plain text that appears on your webpage without being rendered as actual HTML elements.

HTML Entities

  • always start with an ampersand (&) and end with a semicolon (;)
  • can be represented by entity names like &lt; or entity numbers like &#60;
  • entity names are case-sensitive
  • To display a less than sign (<) as text, replace it with &lt;
  • To display a greater than sign (>) as text, replace it with &gt;

Example

To display this code as plain text:     <p> This is a paragraph. </p>

You would write it like this in your HTML:     &lt;p&gt; This is a paragraph. &lt;/p&gt;

Using HTML Entities for the Entities

You can display HTML entity names and numbers as plain text, but you need to use specific encoding to represent the entities themselves, rather than having them interpreted by the browser.

  • For the ampersand (&) use: &amp;
  • For other entities: Replace the ampersand that begins the entity with &amp;

Example

To display:   &lt;

You would write: &amp;lt;

Example

To display:   &#60;

You would write: &amp;#60;

Example

To display:   &amp;

You would write: &amp;amp;
