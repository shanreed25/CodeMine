Indexing

Python indexing refers to the process of accessing individual elements within a sequence (such as a string, list, or tuple) by their position or index number.

Key aspects of Python indexing:

  • Zero-based indexing: Python uses zero-based indexing, meaning the first element in a sequence is at index 0, the second at index 1, and so on.
  • Square bracket notation: Elements are accessed using square brackets [] after the sequence name, with the index number inside the brackets.
  • Positive indexing: Positive indices start from 0 for the first element and increase sequentially for subsequent elements.
  • Negative indexing: Negative indices allow access from the end of the sequence. -1 refers to the last element, -2 to the second to last, and so forth.
  • Immutability: For immutable sequences like strings and tuples, indexing allows accessing elements, but direct modification of individual elements is not permitted. index() method:
  • Mutability: For mutable sequences like lists, indexing can be used to both access and modify elements.

index() method:

Sequences also have an index() method that can be used to find the index of the first occurrence of a specified element within the sequence.

This method can optionally take start and end parameters to limit the search range.
