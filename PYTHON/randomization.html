Randomization

In applications like simulations, games, and testing we may want to use some sort of randomization.

Python's random module allows you to introduce randomness into your programs. It relies on a pseudo-random number generator, which uses an algorithm to produce sequences of numbers that appear random but are actually deterministic given a starting seed.

It provides functions for generating random numbers, selecting random elements from sequences, and shuffling sequences.

Random Module

  • A Python's built-in library for generating random numbers.
  • It provides functions for generating pseudo-random numbers, both integers and floating-point numbers.
  • It also offers functions for selecting random elements from sequences, shuffling sequences, and more.

Random Module Common Functions

  • random.randint(a, b): Returns a random integer between a and b (inclusive)
  • random.random():Returns a random float number between 0.0(inclusive) and 1.0(exclusive)

Title

import random

rand_num = random.randint(1, 10)

print(rand_num)

  • Prints a random whole number between 1 and 10 (inclusive).
