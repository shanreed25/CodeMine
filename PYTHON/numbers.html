String Formatting - f-strings (Formatted String Literals)

This is a modern and readable way to format strings in Python 3.6+

f-string

number = 8.6

formatted_number = f"{number:.2f}"

print(formatted_number)


number = 3.14159

formatted_number = f"{number:.2f}"

print(formatted_number)


number = 5

formatted_number = f"{number:.2f}"

print(formatted_number)

  • Explanation
  • f"..." denotes an f-string.
  • {number:.2f} formats the number variable
  • : introduces the format specifier
  • .2f specifies that the number should be formatted as a fixed-point number (f) with two decimal places (.2). This also handles rounding automatically

format() method

This method is available for string objects and provides similar formatting capabilities

format() method

number = 8.6

formatted_number = "{:.2f}".format(number)

print(formatted_number)


number = 3.14159

formatted_number = "{:.2f}".format(number)

print(formatted_number)


number = 5

formatted_number = "{:.2f}".format(number)

print(formatted_number)


  • "{:.2f}" is the format string.
  • .format(number) applies the format to the number variable
