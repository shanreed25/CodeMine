Functions

In its simplest form is just a wrapper name for a block of code. It is a self-contained block of code designed to perform a specific task

You give it name and then when you call the function by that name, all the code within the function block will be executed.

It can help us save time and reduce repeated code.

All functions have two key steps

  1. Function definition
  2. Invoking/calling the function

Syntax: Defining a Function

def my_function () :
              do this first
              do this second
              do this last

def my_function ( parameter1, parameter2 ) :
              result = parameter1 + parameter2
              return result

  • Functions are defined using the "def" keyword
  • "def" is followed by a function name, parentheses (which may contain parameters) and a colon(:)
  • The function's body is indented below the def line

Calling a Function

def greet ( ) :
        print( "Hello from the hello function!" )
        print( "Goodbye from the hello function!" )

greet ( )

def sayHello ( name ) :
        print( f" Hello, { name } ! " )


sayHello("Shannon")

  • To execute the code within a function, you call it by its name, followed by parentheses, enclosing any necessary arguments
