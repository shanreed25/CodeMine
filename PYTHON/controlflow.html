Control Flow

Control flow in Python dictates the order in which individual statements, instructions, or function calls are executed or evaluated within a program

By default, Python code executes sequentially from top to bottom. However, control flow structures allow for deviation from this linear execution, enabling decision-making, repetition, and error handling.

The primary control flow structures in Python include

  • Conditional Statements (if, elif, else): These statements allow the program to execute different blocks of code based on whether a given condition is true or false.
  • Loops (for, while): Loops enable the repeated execution of a block of code.
  • Exception Handling (try, except, finally, else): These statements manage errors and unexpected events that may occur during program execution.
  • Structural Pattern Matching (match, case): Introduced in Python 3.10, this allows for more complex conditional logic based on the structure of data.
  • Jump Statements(break, continue, pass):

These control flow mechanisms are fundamental to creating dynamic and responsive Python programs, allowing them to adapt to different conditions and inputs.

Conditional Statements

These statements allow the program to execute different blocks of code based on whether a given condition is true or false.

  • if: introduces a conditional statement
  • else: else keyword introduces an alternative code block
  • elif: checks multiple conditions sequentially. It executes the corresponding block of code if its condition is true and the preceding if or elif conditions were false.

Loops

Loops enable the repeated execution of a block of code.

  • for loop: Iterates over a sequence (e.g., a list, tuple, string) or other iterable object, executing a block of code for each item in the sequence.
  • while loop: Repeats a block of code as long as a specified condition is true.

  • pass: Acts as a placeholder. It does nothing, but it's useful when a statement is required syntactically but you don't want any code to be executed.
