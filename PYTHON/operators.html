Logical Operators

Python offers three logical operators: and, or, and not. They are fundamental for creating complex conditional logic in Python, enabling programs to make decisions and control flow based on more than one criteria

These operators, also known as Boolean operators, are used to combine and evaluate conditional statements, determining the overall truth value of an expression.

and Operator:

  • The and operator returns True if both operands (expressions) are True
  • If either or both operands are False, the and operator returns False
  • Example: (x > 5) and (x < 10) would be True only if x is both greater than 5 and less than 10

or Operator:

  • The or operator returns True if at least one of the operands (expressions) is True
  • It returns False only if both operands are False
  • Example: (x < 5) or (x > 10) would be True if x is less than 5 or greater than 10 (or both, though not possible in this specific example)

not Operator:

  • The not operator is a unary operator, meaning it operates on a single operand
  • It reverses the Boolean value of its operand. If the operand is True, not makes it False, and if the operand is False, not makes it True
  • Example: not (x == 0) would be True if x is not equal to 0, and False if x is equal to 0
