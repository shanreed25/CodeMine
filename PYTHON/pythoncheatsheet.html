Basics What It Does How To Use
Print Prints a string into the console print("Hello World")
New Line Represents the newline character. It is an escape sequence used within strings to indicate a line break.
When \n is encountered in a string that is printed or written to a file, it causes the output to move to
the beginning of the next line 		\n
Input Prints a string into the console, and asks the user for a string input input("What's your name\n")
Comments Adding a # symbol in font of text lets you make comments on a line of code, The computer will ignore your comments #This is a comment
Variables A variable give a name to a piece of data. Like a box with a label, it tells you what's inside the box age = 32
The += Operator This is a convient way of saying: "take the previous value and add to it price = 5
