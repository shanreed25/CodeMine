Tip Calculator

To build a basic tip calculator in Python

  1. Get user input for the bill amount, tip percentage, and the number of people that will be spliting the bill
  2. You will need to convert the inputs into the appropiate data type
    • The inputs will return a string but we need to work with it as numbers
    • the bill should be a floating point number
    • the tip percentage and number of people should be a integer
  3. Divide the tip percentage by 100, to convert it to a decimal then add 1 to it

Pyton Tip Calculator

print( "Welcome to the python tip calculator!" )
bill = float(input( "What was the total bill? $" ))
tip_percent = int(input( "What percentage tip would you like to give? 10 12 15 " )) / 100
tip_amount = bill * tip_percent
people = int(input( "How many people to split the bill? " ))
amount = ( bill + tip_amount ) / people
amount = f" { amount : .2f } " # Formats the number to two decimal places and ensure the display includes trailing zeros
print( amount )
