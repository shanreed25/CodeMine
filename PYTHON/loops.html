Loops

Loops enable the repeated execution of a block of code.

  • for loop: Iterates over a sequence (e.g., a list, tuple, string) or other iterable object, executing a block of code for each item in the sequence.
  • while loop: Repeats a block of code as long as a specified condition is true.

For Loop

A For loop is a control flow statement used to iterate over a sequence (like a list, tuple, string, or range) or other iterable objects.

It executes a block of code once for each item in the sequence.

Key characteristics of Python for loops

  • Iteration over iterables: Python's for loops are designed to iterate directly over the elements of an iterable
  • Syntax: for item in iterable:, where item is a variable that takes on the value of each element in iterable during each iteration
  • Code block: The code to be executed repeatedly is defined by indentation following the for statement.

Syntax: for loop

# Iterating through a list

items = [ "item1", "item2", "item3" ]

for item in items_list:
        print(item)


# Iterating through a string

word = "Python"

for char in word:
        print(char)


# Iterating through a range of numbers

for i in range(5):
        print(i)

While Loop

epeatedly executes a block of code as long as a specified condition remains true.

The loop continues to iterate, checking the condition at the beginning of each iteration.

If the condition evaluates to True, the code inside the loop's body is executed.

If the condition evaluates to False, the loop terminates, and program control moves to the statements immediately following the loop.

Syntax: while loop

while condition:
       Code block to be executed repeatedly

(Indented statements)

  • while keyword: Initiates the while loop.
  • condition: A boolean expression that is evaluated at the start of each iteration. The loop continues as long as this condition is True.
  • colon (:): Marks the end of the while statement.
  • Indented code block: The statements that are repeatedly executed as long as the condition is True.

Example: for loop

count = 0 while count < 5: print(f"Count is: {count}") count += 1 print("Loop finished.")

  • the loop continues as long as count is less than 5
  • Inside the loop, the current value of count is printed, and count is incremented by 1 in each iteration
  • Once count reaches 5, the condition count < 5 becomes False, and the loop terminates, executing the "Loop finished." statement.

Remember.....

  • Infinite loops: If the condition never becomes False, the while loop will run indefinitely, creating an infinite loop. This typically requires a break statement or an external interruption to stop execution.
    • break: Terminates the loop immediately, regardless of the condition.
    • continue: Skips the rest of the current iteration and proceeds to the next iteration (re-evaluating the condition).
    • else(optional): A block of code that is executed once after the while loop completes normally (i.e., the condition becomes False), but not if the loop is terminated by a break statement
