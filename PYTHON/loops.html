Loops

Loops enable the repeated execution of a block of code.

  • for loop: Iterates over a sequence (e.g., a list, tuple, string) or other iterable object, executing a block of code for each item in the sequence.
  • while loop: Repeats a block of code as long as a specified condition is true.

For Loop

A For loop is a control flow statement used to iterate over a sequence (like a list, tuple, string, or range) or other iterable objects.

It executes a block of code once for each item in the sequence.

Key characteristics of Python for loops

  • Iteration over iterables: Python's for loops are designed to iterate directly over the elements of an iterable
  • Syntax: for item in iterable:, where item is a variable that takes on the value of each element in iterable during each iteration
  • Code block: The code to be executed repeatedly is defined by indentation following the for statement.

Syntax: for loop

# Iterating through a list

items = [ "item1", "item2", "item3" ]

for item in items_list:
        print(item)


# Iterating through a string

word = "Python"

for char in word:
        print(char)


# Iterating through a range of numbers

for i in range(5):
        print(i)
