IF Statement

Allow a program to make decisions and perform different actions based on whether a given condition is true or false

  • The simplest form checks a single condition.
  • If the condition evaluates to True, the indented block of code following the if statement is executed

Key Elements:

  • Condition: A boolean expression that evaluates to either True or False
  • Colon(:): Required after the condition in if, elif, and else statements
  • Indentation: Python uses indentation (typically 4 spaces) to define code blocks. Code within an if, elif, or else block must be consistently indented.

Conditions can be combined using logical operators

Syntax: if (simplest form)

if condition:
      do this if condition is true

  • if: introduces a conditional statement
  • condition: is an expression that evaluates to either True or False
  • If condition is True, the indented code block immediately following the if statement is executed.

Example: if (simplest form)

age = 28
if age >= 18:
      print( "You are eligible to vote." )

  • condition: age >= 18
  • since age is greater than 18(true) "You are eligible to vote" will be printed

if...else statement

The else keyword allows specifying an alternative block of code to be executed if the if condition evaluates to False

if statement

if condition:
      do this if condition is true
else:
      do this if condition is false

  • If the condition in the if statement evaluates to False, the indented code block immediately following the else statement is executed.

Code Name

age = 15
if age >= 18 :
      print( "You are eligible to vote." )
else :
      print( "You are not eligible to vote." )

  • condition: age >= 18
  • since age is less than 18(false) "You are not eligible to vote" will be printed
