Lists

Built-in data type used to store an ordered collection of items

  • Ordered: items in a list maintain a specific order and order does not change unless explicitly modified
  • Index Ordered: Ordered by index that start at 0
  • Mutable: list are changeable
  • Heterogeneous: the elements can be of different types
  • Created and searched using square brackets[]
  • Allow duplicates

List Index

List are ordered by index that start at 0, meaning index 0 is the first item in the list

You can access items from the end of the list, so -1 will access the last item in the list
