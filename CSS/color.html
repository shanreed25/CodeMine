CSS Color property

Remove or reset the color of an element's text or background using CSS properties

To remove the color of the text within an element and let it revert to its default color (typically black), you can use the color property.

  1. Setting the color property to initial: This sets the text color to the initial value as defined by the browser.
    • selector {color: initial;}
  2. Using the unset keyword: If the color property is inheritable, unset will reset it to the inherited value. If it's not inheritable, it will reset it to the initial value.
    • selector {color: unset;}
  3. Setting the color to a specific value: You can explicitly set the color to a default value like black (#000000).
    • selector {color: #000000; /* or color: black; */}
